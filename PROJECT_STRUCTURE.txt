Backend (Python + FastAPI)
  • main.py          → FastAPI app, scheduler setup, API endpoints
  • database.py      → SQLite (secrets, campaign tables)
  • agents.py        → AI workflow (search, draft, critique, image gen, post)
  • auth.py          → OAuth flow for X/Twitter and LinkedIn

Frontend (Next.js 15 + TailwindCSS)
  • app/page.tsx     → Main UI with connection boxes + prompt box
  • components/      → Reusable UI components
  • next.config.ts   → Proxies /api → localhost:8000

Configuration
  • backend/.env     → (You create) API keys for Gemini, X, LinkedIn
  • .env.example     → Template for environment variables
